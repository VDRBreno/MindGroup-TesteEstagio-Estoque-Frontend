## Instalação ( Yarn )

```bash
git clone https://github.com/VDRBreno/MindGroup-TesteEstagio-Estoque-Frontend.git

cd MindGroup-TesteEstagio-Estoque-Frontend

# Instalar as dependências
yarn

# Iniciar aplicação
yarn dev

# Aplicação rodando em localhost:5173
```
